# 🌍 FairTrip – Tourist Anti-Scam App

FairTrip is a web app that helps tourists avoid getting scammed by showing the **fair price of products and taxi fares**.

## ✨ Features

- 💰 Product Price Checker
- 🚕 Taxi Fare Estimator
- ⚠ Scam Awareness Page
- ⭐ Trusted Shops List

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend (Planned):
- Node.js
- Express
